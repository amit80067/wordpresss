Ultimate Booking Manager is built to be used for booking or reservation functionality. It can be used for any business or niche websites.

== Description ==

This is a Multipurpose “Booking System” plugin can be used by **Medical Doctors, Beauty Salon, Hairdresser & Spa Services, Massage Therapists, Handyman, Cleaning Services and all Contract based Consultants, Training Sessions, Art, Music and Dance Classes** and Hourly or Price Based Service Providers.

**Ultimate Booking** Plugin Provides One Page and Multilevel booking system  for your Website. User can Check availability and Make bookings for your services with different duration and Price and Products Availablity. You can Control the **Staff (Team) or Resource availability** and Services List to be Shown in the Form.

[Demos](http://nichewpthemes.com/bm/) | [HELP](http://support.wedesignthemes.com/forum/wordpress-plugins/ultimate-booking-manager/)

Your website visitors can check availability of services you offer and make a booking for specific Hourly or Multiple hours time slot according to your Services you offer (Ex: Beard Shave - 1 hr, Hair Cut - 2 hrs). Our **Booking  Manager** provides Multiple features to complete almost all niche based Business Services booking  functionality.

**Booking reservations & appointments** with a **Specific time** and a Hourly duration for a service and **Specific Staff**. Each calendar can have multiple services with different  **duration and prices**. 

== Booking Manager Customers will be able to: ==

- Select day(s) in calendar
- Select a Service
- Can Choose the Person or Staff (He/She Desired)
- Select available time-slot to book.
- Fill Essential booking form fields
- Submit a booking

== The Notification Emails ==

**Administrator** and **Staff** also your **visitors** will receive **notification emails**, like new booking and Changes and Cancellations.

Admin can **Approve** or **Decline** the reservations in from **Weekly Calendar** View Booking Panel. Staff Wise Booking List can be Seen too. 

The notification emails can be in **“Plain Text”** format  or in **"Custom HTML”**  formats.

== Booking Manager Benefits: ==
- Simple and easy to use.
- Very flexible functionality. Fit to very wide range of **niche business**.
- All bookings and settings are stored in your **WordPress DB**.
- **Responsive** front end and back end design.
- Customizable email notifications for administrator, Staff and users Form validation and built it **anti-spam captcha** protection 
- An easy to use Booking Admin Panel that displays bookings in Weekly Calendar Overview or **Staff Wise Listing** Table and lets you manages bookings.
- Can redirect customer to the specific **“Thank you”** page, after successful booking process.  
- **WPML** Multi language support.
- Working dates, holiday dates, not **available dates** and specific time can be defined
- Can Control restriction of default, **maximum** and **minimum hours**
- **Open hours** can be defined for each date based on Staff or Service
- Booking calendar can have **multiple services** defined
- All services can have its **own price** and **duration**
- Available times are managed automatically to **avoid double-booking**
- Booking form can be connected with **payment modules** (Ex: PayPal Standard, Pay at arrival (Cash))

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dt-booking-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Services->Settings screen to configure the plugin
4. Choose desired options & save to finish the configuration.

== Frequently Asked Questions ==

= Our Support Forum =

If you have some questions, you can post them at [technical help board](http://support.wedesignthemes.com/forum/wordpress-plugins/ultimate-booking-manager/)

= Multi-site Compatible? =

No, It's only single domain plugin.

= Multi-language Support? =

Yes, It's compatible with [WPML](https://wpml.org/) plugin. You can switch any language easily.

= How many Payment Gateways Support? =

Now it supports only PayPal Payment Gateway. Also "Pay At Arrival" option included.

= Is this plugin require to install Elementor Plugin? =

No, It works with basic setup. No need require to install page builder plugins. But it's compatible with [WP Bakery Plugin](https://wpbakery.com/) and [Elementor Plugin](https://elementor.com/).

= What are all the shortcodes available for appointment? =

* Reservation Form : [dt_sc_reservation_form title="" serviceids="" staffids="" /]
The Reservation Form only displays the available services and staffs. It's does not make any appointment. When you submit this, it's redirected to appointment page.
* Reserve Appointment : [dt_sc_reserve_appointment title="" type="type1, type2" /]
In this shortcode, there are two types of form displaying available. Use any type to make appointment for your need.
